## <a name="effort-tracking-view"></a><span data-ttu-id="b5e83-108">Piepūles izsekošanas skats</span><span class="sxs-lookup"><span data-stu-id="b5e83-108">Effort tracking view</span></span>

<span data-ttu-id="b5e83-109">Skats **Piepūles izsekošana** izseko uzdevumu progresu grafikā, salīdzinot faktiskās ieguldījuma stundas, kas tiek patērētas uzdevumam, ar plānotajām ieguldījuma stundām.</span><span class="sxs-lookup"><span data-stu-id="b5e83-109">The **Effort tracking** view tracks the progress of tasks in the schedule by comparing the actual effort hours spent on a task to the task's planned effort hours.</span></span> <span data-ttu-id="b5e83-110">Lai aprēķinātu izsekošanas metriku, programmatūrā Dynamics 365 Project Operations tiek izmantotas tālāk norādītās formulas.</span><span class="sxs-lookup"><span data-stu-id="b5e83-110">Dynamics 365 Project Operations uses the following formulas to calculate the tracking metrics:</span></span>

## <a name="reprojecting-effort"></a><span data-ttu-id="b5e83-118">Piepūles pārprojektēšana</span><span class="sxs-lookup"><span data-stu-id="b5e83-118">Reprojecting effort</span></span>
