<span data-ttu-id="41f5f-109">Lai atvērtu skatu **Izmaksu uzskaite**, lapas **Projekti** cilnē **Uzskaite** atlasiet **Uzskaite** > **Izmaksas**.</span><span class="sxs-lookup"><span data-stu-id="41f5f-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view.</span></span> <span data-ttu-id="41f5f-110">Vai arī atlasiet **Lietot** > **Rēķina likme**, lai atvērtu skatu **Ieņēmumu uzskaite**, kurā redzami darbaspēka ieņēmumi katrā projekta plāna uzdevumā.</span><span class="sxs-lookup"><span data-stu-id="41f5f-110">Or, you can select **Use** > **Bill Rate** to open the **Revenue Tracking** view, which shows the progress of labor revenue on each task in the project plan.</span></span> <span data-ttu-id="41f5f-111">Šis skats salīdzina arī faktiskos uzdevuma darbaspēka ieņēmumus ar uzdevuma plānotajiem darbaspēka ieņēmumiem.</span><span class="sxs-lookup"><span data-stu-id="41f5f-111">This view also compares the actual labor revenue spent on a task to the task's planned labor revenue.</span></span> <span data-ttu-id="41f5f-112">Lai aprēķinātu darbaspēka ieņēmumu metriku, Project Operations izmanto šādas formulas:</span><span class="sxs-lookup"><span data-stu-id="41f5f-112">Project Operations uses the following formulas to calculate labor revenue metrics:</span></span>
