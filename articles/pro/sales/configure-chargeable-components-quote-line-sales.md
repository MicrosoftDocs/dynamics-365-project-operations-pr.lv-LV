  - <span data-ttu-id="dfc35-113">Nav iekļaujams rēķinā</span><span class="sxs-lookup"><span data-stu-id="dfc35-113">Non-chargeable</span></span>

### <a name="update-a-project-task-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="dfc35-121">Projekta uzdevuma atjaunināšana par apmaksājamu vai neapmaksājamu</span><span class="sxs-lookup"><span data-stu-id="dfc35-121">Update a project task to be chargeable or non-chargeable</span></span>

<span data-ttu-id="dfc35-122">Projekta uzdevums var būt apmaksājams vai neapmaksājam noteiktā projekta piedāvājuma rindā, kas nodrošina šādas iestatīšanas iespējas.</span><span class="sxs-lookup"><span data-stu-id="dfc35-122">A project task can be chargeable or non-chargeable in the context of a specific project-based quote line, which makes the following setup possible:</span></span>
